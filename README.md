# fusion-communes

## Installation

    git clone git@github.com:cquest/fusion-communes.git
    cd fusion-communes
    pip install -r requirements.txt

## Utilisation

### 2017

    Script de récupération des données wikipédia:

    ./2017/wikipedia2json.py > fusion2017.json


### 2016

    ./fusion2016.py NUM_DEPARTEMENT
